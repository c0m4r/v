# Project description

V - Lightweight KVM virtual machine manager for Linux
Provides a CLI and web UI for creating, running, and managing QEMU/KVM virtual machines with cloud-init support.

# Project structure

- engine: ./engine/
- cli: ./cmd/cli/
- web: ./cmd/web/

# Rules

1. Test and lint with ./addons/check.sh
2. Build with ./addons/build.sh
